DanmakuFlameMaster
==================

android上的弹幕转换、渲染输出源码


### Features

- 使用SurfaceView高效绘制

- A站json弹幕格式解析

- B站xml弹幕格式解析

- 基础弹幕精确还原绘制

- 多核机型优化，可选独立线程缓存绘制

- 支持多种绘制选项动态切换
